# Configuration

Sythoria is designed to work out of the box, but offers several customization options.

## Model selection

Choose your model from the dropdown in the chat input bar. Available models depend on your selected provider and API key.

## Temperature

Control the randomness of responses via the **Temperature** slider in settings:

- **0.0** — Deterministic, focused outputs
- **0.7** — Balanced creativity (default)
- **2.0** — Maximum randomness

Lower temperatures are better for factual tasks. Higher temperatures work well for creative writing.

## System prompts

Sythoria includes built-in system prompts you can select, or write your own:

1. Open **Settings** from the sidebar
2. Navigate to the **System Prompt** section
3. Choose a preset or write a custom prompt

System prompts set the behavior and personality for the AI assistant in each conversation.

## Provider configuration

Switch providers at any time from the chat input bar. Each provider may have specific settings:

- **API Base URL** — The endpoint for API calls
- **API Key** — Your authentication key (stored locally only)
- **Default Model** — The model to use when creating new conversations

See the [Providers](/docs/providers/openai) section for provider-specific guides.

## Keyboard shortcuts

| Shortcut      | Action           |
| ------------- | ---------------- |
| `Enter`       | Send message     |
| `Shift+Enter` | New line         |
| `Escape`      | Stop streaming   |
| `⌘K / Ctrl+K` | Search (in docs) |

## Data management

All data is stored in your browser. You can:

- **Export conversations** — Click the export button in the sidebar menu
- **Clear all data** — Clear browser storage for the site to reset everything
