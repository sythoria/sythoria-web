# Quickstart

Sythoria is a lightweight, privacy-first AI chat interface that connects to any OpenAI-compatible API. No accounts, no tracking — just pick a provider, add your key, and start chatting.

## Choose how to use Sythoria

### In your browser

1. Go to [sythoria.app/chat](/chat)
2. Click **Start** on the welcome screen
3. Open **Settings** (gear icon in the sidebar) and configure your first model
4. You're ready to chat

### As a desktop app

1. Download the latest release from [GitHub](https://github.com/sythoria/sythoria-desktop/releases)
2. Install and launch Sythoria
3. Click **Start** on the welcome screen
4. Open **Settings** and configure your first model
5. You're ready to chat

> Both versions work identically. Your data stays local in either case — the desktop app just gives you a dedicated window and OS integration.

## Add your first model

When you open Settings for the first time, a default OpenAI model is pre-filled. You can keep it, switch providers, or add more. For each model you need a provider preset and an API key — the rest fills in automatically.

| Provider      | Setup guide                                              | Where to get your key                                                | Key? |
| ------------- | -------------------------------------------------------- | -------------------------------------------------------------------- | ---- |
| OpenAI        | [/docs/providers/openai](/docs/providers/openai)         | [platform.openai.com/api-keys](https://platform.openai.com/api-keys) | Yes  |
| Anthropic     | [/docs/providers/anthropic](/docs/providers/anthropic)   | [console.anthropic.com](https://console.anthropic.com/)              | Yes  |
| Google Gemini | [/docs/providers/gemini](/docs/providers/gemini)         | [aistudio.google.com/apikey](https://aistudio.google.com/apikey)     | Yes  |
| Ollama        | [/docs/providers/ollama](/docs/providers/ollama)         | None — just run `ollama server`                                      | No   |
| OpenRouter    | [/docs/providers/openrouter](/docs/providers/openrouter) | [openrouter.ai/keys](https://openrouter.ai/keys)                     | Yes  |
| NVIDIA NIM    | [/docs/providers/nvidia-nim](/docs/providers/nvidia-nim) | [build.nvidia.com](https://build.nvidia.com/)                        | Yes  |

Your API key is stored **only in your browser's localStorage** — it never touches Sythoria's servers. See [Privacy & Security](/docs/privacy) for the full breakdown.

## Start chatting

1. Pick a model from the **dropdown in the chat input bar**
2. Type a message and press `Enter`
3. Responses stream in real-time, token by token

That's it. You're running Sythoria.
