# Quickstart

Sythoria is a lightweight, privacy-first AI chat interface that works with any OpenAI-compatible API. No accounts, no tracking — just chat.

## 1. Open Sythoria

Navigate to [sythoria.app/chat](/chat) or run the app locally.

## 2. Enter your API key

On first launch, you'll be prompted to enter an API key. Your key is stored **only in your browser** — it never touches our servers.

- For **OpenAI**, use an API key from [platform.openai.com](https://platform.openai.com/api-keys)
- For **Anthropic**, use a key from [console.anthropic.com](https://console.anthropic.com/)
- For **Ollama**, no key is needed — just have Ollama running locally on port 11434

## 3. Select a model

Pick a model from the dropdown in the chat input bar. Sythoria auto-detects available models based on your provider.

## 4. Start chatting

That's it. Type a message and hit Enter. Responses stream in real-time.
