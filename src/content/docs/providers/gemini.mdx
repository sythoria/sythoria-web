# Google Gemini

Connect Sythoria to Google's Gemini API.

## Setup

1. Go to [aistudio.google.com/apikey](https://aistudio.google.com/apikey)
2. Create an API key
3. In Sythoria, select **Google Gemini** as the provider
4. Paste your API key when prompted

## Default configuration

- **API Base:** `https://generativelanguage.googleapis.com/v1beta/openai/chat/completions`
- **Default Model:** `gemini-2.5-pro`

## Available models

- `gemini-2.5-pro` — Most capable Gemini model
- `gemini-2.5-flash` — Fast and efficient
- `gemini-2.0-flash` — Previous generation flash model

## Notes

Sythoria uses Google's OpenAI-compatible endpoint, so the experience is identical to using OpenAI's API.

## Troubleshooting

**403 Forbidden:** Your API key may not have access to the requested model. Check your Google Cloud project settings.
