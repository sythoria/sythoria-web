# OpenAI

Connect Sythoria to OpenAI's API to access their complete suite of frontier models, including the flagship GPT-5.6 series (Sol, Terra, Luna), the general-purpose GPT-5, the advanced reasoning-driven `o` series (o1, o1-mini, o3-mini), and the highly efficient GPT-4o models. OpenAI offers the industry-standard developer platform with strong general-purpose reasoning, state-of-the-art coding capabilities, and vision processing.

## Prerequisites

- An OpenAI developer account with billing enabled
- An API key generated from the OpenAI developer platform

## Setup

1. Log in to your OpenAI developer account and navigate to [platform.openai.com/api-keys](https://platform.openai.com/api-keys).
2. Click **Create new secret key**, give it an optional name (e.g., "Sythoria Desktop"), and copy it.
3. In Sythoria, open **Settings** from the sidebar.
4. Select **OpenAI** from the **Provider Preset** dropdown — the API base URL and default model will fill in automatically.
5. Paste your API key into the **API Key** field.
6. The connection indicator should turn **green** within a few seconds, showing that Sythoria has successfully authenticated.

> [!NOTE]
> Your API key is stored securely in your OS keychain. It is never sent to Sythoria's servers. See [Privacy & Security](/docs/privacy).

## Default configuration

| Setting           | Value                                        |
| ----------------- | -------------------------------------------- |
| **API Base**      | `https://api.openai.com/v1/chat/completions` |
| **Default Model** | `gpt-5.6-terra`                              |
| **API Format**    | OpenAI Chat Completions                      |
| **Streaming**     | Supported                                    |

### Reasoning Models (o1, o1-mini, o3-mini)

OpenAI's reasoning models are designed to think step-by-step before returning a response, making them exceptional at complex logic, math, and software engineering.

- **Reasoning Tokens**: These models generate internal "reasoning tokens" that represent their thought process. These tokens count toward your output limits and billing but are hidden from the final chat display.
- **Auto-adjusted Parameters**: Certain parameters (like custom temperature and frequency penalties) are not supported by the OpenAI API for reasoning models. Sythoria automatically adjusts or strips these parameters dynamically when a reasoning model is selected to prevent API errors.

### Multimodal Capabilities

Models like `gpt-4o`, `gpt-4o-mini`, `gpt-5`, and the `gpt-5.6` series support image inputs. You can drag and drop images directly into Sythoria's chat input or take screenshots to analyze visual data, UI designs, or charts.

## Pricing

OpenAI charges per token. Rates vary by model, with lighter models like `gpt-4o-mini` costing significantly less than flagship models like `gpt-5.6-sol`. Check [openai.com/pricing](https://openai.com/pricing) for the latest rates.

## Switching models mid-conversation

You can switch to a different OpenAI model at any time using the model dropdown in the chat input bar. Each new message uses the currently selected model, allowing you to start a session with a fast model like `gpt-5.6-luna` or `gpt-4o-mini`, and switch to a more capable reasoning model like `gpt-5.6-sol` or `o3-mini` when facing a challenging question — all within the same chat history.

## Troubleshooting

| Error                               | Cause                                                                   | Fix                                                                                                                                                                  |
| :---------------------------------- | :---------------------------------------------------------------------- | :------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **401 Unauthorized**                | API key is invalid, revoked, or expired                                 | Generate a new API key at [platform.openai.com/api-keys](https://platform.openai.com/api-keys) and verify it was pasted correctly.                                   |
| **429 Rate Limit / Quota Exceeded** | You have reached your usage tier limit or ran out of account credits    | Check your account limits and credit balance at [platform.openai.com/settings/organization/billing](https://platform.openai.com/settings/organization/billing).      |
| **Model not found / 404**           | The selected model is not available to your API key or usage tier       | Verify the model ID at [platform.openai.com/models](https://platform.openai.com/models). Note that newer models may require a higher API usage tier (e.g., Tier 1+). |
| **402 Payment Required**            | No billing method is on file or the account has zero pre-funded credits | Add funds or update your payment method at [platform.openai.com/settings/organization/billing](https://platform.openai.com/settings/organization/billing).           |
| **Connection indicator red**        | Network issue, local firewall block, or invalid API Base URL            | Verify that your network is connected and that the API Base URL is set to `https://api.openai.com/v1/chat/completions`. Click **Refresh** to re-test the connection. |
